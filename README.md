# 🌐 We Are Five Marketing Agency

**We Are Five** is a modern, responsive marketing agency website designed to present digital services with a clean, professional UI and smooth user experience.  
The project focuses on strong branding, performance optimization, and a modern web presence.

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Clean and modern UI/UX
- Smooth navigation and user interactions
- Fast loading and performance optimized
- Professional branding layout

- HTML5  
- CSS3  
- JavaScript  
- Responsive Design Principles

## 📌 Purpose

This project is built to demonstrate a real-world marketing agency website with modern design principles and production-level UI structure.
